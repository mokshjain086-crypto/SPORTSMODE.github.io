# 🏀 SPORTS MODE - Premium Sports Equipment Website

## 📌 Overview
SPORTS MODE is a modern and responsive e-commerce website designed for selling premium sports equipment.  
It provides a clean user interface where users can browse products, explore categories, and interact with a visually appealing shopping experience.

---

## 💡 Solution
This project solves the problem of creating a **simple and attractive online sports store UI** without using heavy frameworks.

It:
- Displays featured sports products with pricing and descriptions  
- Organizes items into categories for easy browsing  
- Provides a responsive design for desktop and mobile  
- Enhances user experience with smooth animations and modern UI  

---

## 🔍 Key Features
- ✅ Responsive design (mobile + desktop)
- ✅ Hero section with call-to-action  
- ✅ Product listing with cards  
- ✅ Category-based browsing  
- ✅ Smooth scrolling navigation  
- ✅ Modern UI with hover effects  

---

## 🛠️ Technology Stack

### Frontend
- HTML5  
- CSS3 (Flexbox + Grid + Animations)  
- JavaScript (Vanilla JS)

### Styling
- Custom CSS (No frameworks used)  
- Responsive design using media queries  

### Assets
- SVG-based placeholder images  
- Icons using emojis
